    モーター確認

    モーター（ステアリング・スロットル）の動作確認を行います。

    モーターの値を確認する

    信号値の数値を入れてEnterを押す：

    python motor.py

    ステアリングのPWM値を探す

    1. 真ん中 - ステアリングがまっすぐになる値
    2. 左最大 - 左いっぱいに切った値
    3. 右最大 - 右いっぱいに切った値

    スロットルのPWM値を探す

    1. ニュートラル - モータードライバーがピッピッピとなる値
    2. 前進の最大値 - 音が変わらなくなるところ
    3. 後進の最大値 - 音が変わらなくなるところ

    注意

    極端に大きな/小さな値を入れるとモーターを破損する恐れがあるため注意！

    RC用ESC（モータードライバー）の仕様

    • 多くのRC用ESCは、前進状態から直接バックできません
    • バックするには、一度ニュートラルに戻してから後進のPWM値を入力
    • ESCの取扱説明書を確認してください

    config.pyに値を保存する

    調整した値をconfig.pyに保存します：

    ## ステアリングのPWM値
STEERING_CENTER_PWM = 370
STEERING_WIDTH_PWM = 80
STEERING_RIGHT_PWM = STEERING_CENTER_PWM + STEERING_WIDTH_PWM
STEERING_LEFT_PWM = STEERING_CENTER_PWM - STEERING_WIDTH_PWM

## スロットルのPWM値
THROTTLE_STOPPED_PWM = 370
THROTTLE_FORWARD_PWM = 500
THROTTLE_REVERSE_PWM = 300

    トラブルシューティング

    モーターが動かない

    1. 電源が入っているか確認
    2. PWM値が適切な範囲内か確認
    3. ESCのキャリブレーションが必要な場合がある

    ステアリングが逆に動く

    STEERING_WIDTH_PWMの符号を反転： 

    STEERING_WIDTH_PWM = -80

    スロットルが反応しない

    1. ESCがニュートラル位置を認識しているか確認
    2. THROTTLE_STOPPED_PWMの値を微調整

    開発者向け: コード解説

    PCA9685によるPWM出力

    PCA9685はI2C接続の16チャンネルPWMコントローラです。

    PCA9685 クラス

    import smbus2

class PCA9685:
    """PCA9685 PWMコントローラ制御クラス"""

    # レジスタアドレス
    MODE1 = 0x00
    PRESCALE = 0xFE
    LED0_ON_L = 0x06

    def __init__(self, i2c_bus: int = 1, address: int = 0x40):
        """
        Args:
            i2c_bus: I2Cバス番号（RPi: 1, Jetson: 7）
            address: I2Cアドレス（デフォルト: 0x40）
        """
        self.bus = smbus2.SMBus(i2c_bus)
        self.address = address
        self._init_device()

    def _init_device(self):
        """デバイス初期化（50Hz設定）"""
        # スリープモードに設定
        self.bus.write_byte_data(self.address, self.MODE1, 0x10)
        # プリスケーラ設定（50Hz: 25MHz / (4096 * 50) - 1 ≒ 121）
        self.bus.write_byte_data(self.address, self.PRESCALE, 121)
        # 通常モードに復帰
        self.bus.write_byte_data(self.address, self.MODE1, 0x00)
        time.sleep(0.005)

    def set_pwm(self, channel: int, value: int):
        """
        指定チャンネルにPWM値を設定

        Args:
            channel: チャンネル番号（0-15）
            value: PWM値（0-4095）。サーボ用は通常200-500程度
        """
        value = max(0, min(4095, value))
        reg = self.LED0_ON_L + 4 * channel
        self.bus.write_byte_data(self.address, reg, 0)       # ON_L
        self.bus.write_byte_data(self.address, reg + 1, 0)   # ON_H
        self.bus.write_byte_data(self.address, reg + 2, value & 0xFF)  # OFF_L
        self.bus.write_byte_data(self.address, reg + 3, value >> 8)    # OFF_H

    PWM値とパルス幅の関係

    PCA9685は周期を4096分割

50Hz = 20ms周期
1カウント = 20ms / 4096 ≒ 4.88μs

PWM値370 → 370 × 4.88μs ≒ 1.8ms（中央）
PWM値290 → 290 × 4.88μs ≒ 1.4ms（左最大）
PWM値450 → 450 × 4.88μs ≒ 2.2ms（右最大）

    正規化値からPWMへの変換

    Actuator クラス

    class Actuator:
    """アクチュエータ制御クラス（ステアリング・スロットル）"""

    def __init__(self, pca9685: PCA9685, channel: int,
                 center_pwm: int, width_pwm: int):
        """
        Args:
            pca9685: PCA9685インスタンス
            channel: PWMチャンネル番号
            center_pwm: 中央位置のPWM値
            width_pwm: 中央から最大までの振れ幅
        """
        self.pca9685 = pca9685
        self.channel = channel
        self.center_pwm = center_pwm
        self.width_pwm = width_pwm

    def set_value(self, normalized_value: float):
        """
        正規化値（-1〜1）を設定

        Args:
            normalized_value: -1.0（左/後退）〜 1.0（右/前進）
        """
        # 範囲制限
        normalized_value = max(-1.0, min(1.0, normalized_value))

        # 線形変換: PWM = center + (value × width)
        pwm_value = int(self.center_pwm + (normalized_value * self.width_pwm))

        self.pca9685.set_pwm(self.channel, pwm_value)

    def value_to_pwm(self, normalized_value: float) -> int:
        """
        正規化値をPWM値に変換（デバッグ用）

        Args:
            normalized_value: -1.0〜1.0

        Returns:
            int: PWM値
        """
        return int(self.center_pwm + (normalized_value * self.width_pwm))

    変換例

    正規化値 計算式（center=370, width=80） PWM値
    -1.0（左最大） 370 + (-1.0 × 80) 290
    -0.5（左半分） 370 + (-0.5 × 80) 330
    0.0（中央） 370 + (0.0 × 80) 370
    0.5（右半分） 370 + (0.5 × 80) 410
    1.0（右最大） 370 + (1.0 × 80) 450

    ESCアーミングシーケンス

    ESC（Electronic Speed Controller）は起動時にニュートラル位置を認識する必要があります。

    ESC クラス

    class ESC:
    """ESC制御クラス"""

    def __init__(self, pca9685: PCA9685, channel: int,
                 neutral_pwm: int, forward_pwm: int, reverse_pwm: int):
        """
        Args:
            pca9685: PCA9685インスタンス
            channel: PWMチャンネル番号
            neutral_pwm: ニュートラル位置のPWM値
            forward_pwm: 前進最大のPWM値
            reverse_pwm: 後退最大のPWM値
        """
        self.pca9685 = pca9685
        self.channel = channel
        self.neutral_pwm = neutral_pwm
        self.forward_pwm = forward_pwm
        self.reverse_pwm = reverse_pwm
        self.is_armed = False

    def arm(self) -> bool:
        """
        ESCをアーミング（初期化）

        Returns:
            bool: アーミング成功時True
        """
        # ニュートラル位置を送信してESCを初期化
        self.pca9685.set_pwm(self.channel, self.neutral_pwm)
        time.sleep(2.0)  # ESCがニュートラルを認識するまで待機

        # ピッピッピ音が鳴ればアーミング完了
        self.is_armed = True
        return True

    def set_throttle(self, value: float):
        """
        スロットル値を設定

        Args:
            value: -1.0（後退）〜 0.0（停止）〜 1.0（前進）
        """
        if not self.is_armed:
            raise RuntimeError("ESC is not armed")

        value = max(-1.0, min(1.0, value))

        if value >= 0:
            # 前進: neutral → forward の範囲
            pwm = self.neutral_pwm + int(value * (self.forward_pwm - self.neutral_pwm))
        else:
            # 後退: neutral → reverse の範囲
            pwm = self.neutral_pwm + int(value * (self.neutral_pwm - self.reverse_pwm))

        self.pca9685.set_pwm(self.channel, pwm)

    アーミングシーケンス

    電源ON → ニュートラルPWM送信 → 2秒待機 → ピッピッピ音 → 操作可能

注意: 多くのRC用ESCは前進→後退の直接切り替え不可
      前進 → ニュートラル → 後退 の順序が必要
