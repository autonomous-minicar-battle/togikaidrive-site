    センサー確認

    超音波センサおよびLiDARの動作確認を行います。

    超音波センサの値を確認する

    python ultrasonic.py

    デバイス自動対応

    ultrasonic.pyは自動的にデバイスを検出し、適切なGPIOライブラリを選択します：

    デバイス 使用ライブラリ
    Raspberry Pi 4以前 RPi.GPIO
    Raspberry Pi 5 gpiozero（lgpioバックエンド）
    Jetson Orin Nano Jetson.GPIO

    測定テストの項目

    基本動作確認

    • 各センサーが正しく接続されているか確認
    • 測定値が表示されることを確認

    精度テスト

    • 定規で実際の距離を測り、測定値との誤差を確認（±3mm以内が目安）
    • 異なる距離（10cm、50cm、100cm、200cm）での測定精度を検証

    検知範囲テスト

    • 最小検知距離（約2cm）の確認
    • 最大検知距離（設定値まで）の確認
    • 検知角度（約±15度）を手をかざして確認

    材質による影響

    • 硬い平面（壁、板）での測定
    • 柔らかい素材（布、スポンジ）での測定
    • 角度のある面での反射確認

    やってみよう！

    • 定規で距離を測り、測定値との比較をする
    • 超音波センサの測定可能範囲（角度）を手をかざして調べる
    • 超音波センサの数を変える
    • サンプリングサイクルを変える

    LiDARの値を確認する

    LiDARセンサの動作確認を行います。

    python lidar.py

    ブラウザで http://localhost:8080 にアクセスすると、LiDARビジュアライザーが表示されます。点群データやゾーン分割がリアルタイムで可視化されるので、センサーの取り付け向きや検出範囲が正しいか確認してください。

    ゾーン分割

    LiDARの点群データは超音波センサーと同じ5つのゾーンに分割して使用できます。

                 FrFR 前方
               ↑
    左前 FrLH       FrRH 右前
    　　　↖        ↗ 
RrLH 左   ←   [車]  → RrRH 右
    ゾーン名 説明
    RrLH 左
    FrLH 左前方
    FrFR 前方
    FrRH 右前方
    RrRH 右

    ゾーン判定の計算ロジック

    LiDARの点群データ（角度と距離）から、各点がどのゾーンに属するかを判定します。

    # config.py でのゾーン角度設定例
ZONE_INDEX = {
    "RrLH": (180, 225),   # 左: 180°〜225°
    "FrLH": (225, 315),   # 左前方: 225°〜315°
    "FrFR": (315, 45),    # 前方: 315°〜45°（0度をまたぐ）
    "FrRH": (45, 135),    # 右前方: 45°〜135°
    "RrRH": (135, 180),   # 右: 135°〜180°
}

    各ゾーンの最小距離を取得

    def get_zone_distances(points: list[tuple[float, float]]) -> dict[str, float]:
    """
    点群から各ゾーンの最小距離を計算

    Args:
        points: [(角度, 距離), ...] のリスト

    Returns:
        dict: {"FrFR": 500.0, "FrLH": 300.0, ...}
    """
    zone_distances = {zone: float('inf') for zone in ZONE_INDEX}

    for angle, distance in points:
        if distance <= 0:
            continue

        for zone_name, (start, end) in ZONE_INDEX.items():
            if is_in_zone(angle, start, end):
                # 同じゾーン内で最も近い距離を採用
                zone_distances[zone_name] = min(zone_distances[zone_name], distance)
                break

    return zone_distances

    開発者向け: コード解説

    超音波センサーの距離計測

    超音波センサーはToF（Time of Flight）方式で距離を計測します。

    Ultrasonic クラス

    class Ultrasonic:
    """超音波センサー制御クラス"""

    def __init__(self, trigger_pin: int, echo_pin: int):
        """
        Args:
            trigger_pin: トリガー信号を送るGPIOピン番号
            echo_pin: エコー信号を受けるGPIOピン番号
        """
        self.trigger_pin = trigger_pin
        self.echo_pin = echo_pin

    def measure_distance(self) -> float:
        """
        距離を測定する

        Returns:
            float: 測定距離（mm）。タイムアウト時は config.CUTOFF_RANGE を返す
        """
        # トリガーパルス送信（10μs）
        GPIO.output(self.trigger_pin, True)
        time.sleep(0.00001)
        GPIO.output(self.trigger_pin, False)

        # エコー信号の立ち上がり/立ち下がりを検出
        start_time = time.time()
        while GPIO.input(self.echo_pin) == 0:
            start_time = time.time()

        end_time = time.time()
        while GPIO.input(self.echo_pin) == 1:
            end_time = time.time()

        # ToF計算: 距離 = (時間 × 音速) / 2
        elapsed_time = end_time - start_time
        distance_mm = (elapsed_time * 340000) / 2  # 音速340m/s = 340000mm/s

        return distance_mm

    距離計算の原理

    トリガー  ──┐  ┌──────────────────────────
            │  │
            └──┘  10μs

エコー    ────────┐              ┌────────
                 │    往復時間   │
                 └──────────────┘

距離 = (往復時間 × 音速) / 2
     = (elapsed_time × 340m/s) / 2

    LiDARデータ処理

    LidarProcessor クラス

    class LidarProcessor:
    """LiDAR点群データ処理クラス"""

    def __init__(self, zone_config: dict):
        """
        Args:
            zone_config: ゾーン設定 {"FrFR": (315, 45), "FrLH": (270, 315), ...}
        """
        self.zone_config = zone_config

    def parse_scan_data(self, raw_data: bytes) -> list[tuple[float, float]]:
        """
        生データをパースして点群に変換

        Args:
            raw_data: LiDARからの生バイナリデータ

        Returns:
            list[tuple[float, float]]: [(角度(deg), 距離(mm)), ...] のリスト
        """
        points = []
        # データフォーマットに応じたパース処理
        # YDLIDAR: 各点は角度(2byte) + 距離(2byte)
        for i in range(0, len(raw_data), 4):
            angle = struct.unpack('<H', raw_data[i:i+2])[0] / 100.0
            distance = struct.unpack('<H', raw_data[i+2:i+4])[0]
            points.append((angle, distance))
        return points

    def assign_to_zones(self, points: list[tuple[float, float]]) -> dict[str, float]:
        """
        点群をゾーンに分類し、各ゾーンの最小距離を返す

        Args:
            points: [(角度, 距離), ...] のリスト

        Returns:
            dict[str, float]: {"FrFR": 500.0, "FrLH": 300.0, ...}
        """
        zone_distances = {zone: float('inf') for zone in self.zone_config}

        for angle, distance in points:
            if distance <= 0:
                continue

            for zone_name, (start_angle, end_angle) in self.zone_config.items():
                if self._is_in_zone(angle, start_angle, end_angle):
                    zone_distances[zone_name] = min(zone_distances[zone_name], distance)
                    break

        return zone_distances

    def _is_in_zone(self, angle: float, start: float, end: float) -> bool:
        """角度がゾーン範囲内かを判定（0度をまたぐ場合も対応）"""
        if start <= end:
            return start <= angle < end
        else:  # 0度をまたぐ場合（例: 315〜45）
            return angle >= start or angle < end

    センサーデータの正規化

    normalize_distance 関数

    def normalize_distance(distance_mm: float, max_range: float = 2000.0) -> float:
    """
    距離値を0〜1の範囲に正規化

    Args:
        distance_mm: 距離（mm）
        max_range: 最大測定範囲（mm）。デフォルト2000mm

    Returns:
        float: 正規化された距離（0.0〜1.0）
    """
    normalized = distance_mm / max_range
    return max(0.0, min(1.0, normalized))


def normalize_sensor_data(sensor_data: dict[str, float]) -> dict[str, float]:
    """
    センサーデータ辞書を一括正規化

    Args:
        sensor_data: {"FrFR": 500, "FrLH": 300, ...}

    Returns:
        dict[str, float]: {"FrFR": 0.25, "FrLH": 0.15, ...}
    """
    return {
        zone: normalize_distance(dist)
        for zone, dist in sensor_data.items()
    }
    入力値 max_range=2000 説明
    0mm 0.0 最小距離
    500mm 0.25 近距離
    1000mm 0.5 中距離
    2000mm 1.0 最大距離
